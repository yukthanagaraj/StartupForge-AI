# StartupForge-AI
StartupForge AI is a multi-agent startup planning platform that helps users generate startup ideas, market research, branding, business plans, and roadmaps using AI agents. 

StartupForge AI is a multi-agent startup planning platform that helps founders and innovators transform startup ideas into structured business strategies using specialized AI agents.

The platform uses multiple collaborative agents to generate startup ideas, perform market analysis, create branding strategies, build business plans, design roadmaps, and simulate startup team discussions.

##Features

* Multi-Agent AI Architecture
* Startup Idea Generator Agent
* Market Research Agent
* Branding Agent
* Business Plan Agent
* Roadmap Agent
* Simulation Mode (Virtual Startup Team)
* Persistent Memory System
* Saved Reports Vault
* PDF Export Support
* Markdown Export Support
* Chat History Memory
* Startup Report Generation
* Groq API Integration
* Modular Project Structure

##  Installation

1. Clone repository

git clone YOUR_REPOSITORY_LINK

2. Move into project folder

cd StartupForge-AI

3. Install dependencies

pip install -r requirements.txt

4. Create environment file

copy .env.example .env

5. Add Groq API key

GROQ_API_KEY=your_key_here

6. Run project

streamlit run app.py
